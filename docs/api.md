# Python API

To generate docs, please use:

```bash
cd docs
bash makedocs.sh local-only
```

There are four packages shipped with Jina:

- `jina`: the framework;
- `daemon`: a simple orchestrator for Jina;
- `cli`: the command line interface for Jina.
